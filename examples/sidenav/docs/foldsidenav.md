## 折叠侧栏

### 效果
向下滚动页面，可观察到侧栏是默认折叠的，向下滚动时侧栏对应标题展开。

### 关键配置
`foldsidenav` 可全局配置亦可单页配置，单页配置优先级更高。
```
"options": { // 此 Page 用的编译器的配置
    "foldsidenav": true
}
```
